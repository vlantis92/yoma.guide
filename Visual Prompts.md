# 

Yoma Guide — Visual Style Prompt

Apply this style to all image prompts from the Image Prompts Register.

**Format:** Clean, flat vector-style diagrams. No gradients, shadows, drop-shadows, or 3D effects. No photographic or illustrative elements. White background.

**Colour palette (use these only):**

| Colour    | Hex     | Use for                                           |
| --------- | ------- | ------------------------------------------------- |
| Deep teal | #387f6a | Primary flows, approved states, positive outcomes |
| Amber     | #f0a02d | Highlights, callouts, decision points             |
| Purple    | #41204b | Headers, labels, category groupings               |
| Red       | #fc4d56 | Warnings, rejected states, blockers               |
| Blue      | #51afe9 | Secondary flows, informational elements, links    |
| Black     | #020304 | Body text, lines, borders                         |
| White     | #ffffff | Backgrounds, contrast text on dark fills          |

Half-tone (50% opacity) versions of any colour are permitted for backgrounds, secondary fills, and inactive states.

**Typography:** Sans-serif. Bold for headings and labels. Regular weight for descriptions. All text must be legible at the size the diagram will be displayed.

**Layout:** Left-to-right for linear flows. Top-to-bottom for hierarchies. Generous whitespace. Rounded corners on boxes (small radius). Thin connecting lines (1–2px) with simple arrowheads.

**Icons:** Simple, minimal line icons where needed. Single colour. No filled/detailed illustrations.
